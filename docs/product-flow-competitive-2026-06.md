# Прикорм / Pudding — флоу-аудит + конкурентный анализ

> Метод: 2 параллельные разведки (конкуренты через веб + аудит наших флоу по коду).
> Рамка оценки: «чистый дневник, родитель сам решает, мы НЕ советуем». Дата: 2026-06.

## Главный вывод

**Активной петли поддержания толерантности к аллергену (счётчик «давно не давали» + пуш «пора освежить») не делает НИ ОДИН софтовый конкурент** (проверено 20+ продуктов). Ближайший — Solid Starts — делает это пассивно (подсказка в меню-планере, без пушей) и в РФ-сторе только на английском. Физические продукты (Ready Set Food, Lil Mixins) владеют нарративом «давай 2–3×/нед», но через пакетики, без напоминалок.

→ **Ниша свободна — это заголовок продукта.** Риск: Solid Starts (клиническая команда) может добавить петлю тривиально. Надо успеть.

Иск против **SpoonfulOne** (закрылись в 2022 из-за класс-экшна о дозировках) подтверждает: позиция «дневник без советов/доз» — ещё и юридический щит. Не сворачивать.

## Где выигрываем / отстаём

**Выигрываем:** killer-петля (уникальна), RU+EN (Solid Starts EN-only в РФ), приватность/офлайн, стабильность (TryBaby 3.6★, трекер Solid Starts — самая ругаемая часть лидера).

**Отстаём (table stakes):** нет мульти-кейгивер/облака (Nara/Baby Daybook/Wachanga имеют — главный разрыв; контр-аргумент: приватность), нет PDF-экспорта врачу, нет Watch/виджетов/Siri, ноль установок.

**Не догонять (и не нужно):** контент-моат Solid Starts (видео, нарезка по возрасту) — это «советующие» фичи вне нашей позиции.

## Что чинить в наших флоу (приоритет)

**P1:**
1. **Планирование — «фича-призрак».** Вход только: Календарь → тап по будущему дню. Никто не найдёт. → точка входа на Главной/в Каталоге + аффорданс.
2. **Нет экспорта дневника** (в коде ноль). Главный JTBD-выхлоп + первый Pro-хук.

**P2:**
3. **Главная = «долги», не дневник.** Нет «сегодня записано» и «запланировано на сегодня».
4. **«Дал» пишет representative группы, а не выбранный продукт** (дал сыр → записан йогурт). Нужен выбор + бэкдейт.
5. **Онбординг:** имя зря обязательно (SPEC: опц.), нет «Назад», «уже ввели» — плоская сетка 71 без поиска/категорий; шаг недоступен после онбординга.
6. **Уведомления — тихая смерть killer-фичи** при отказе: нет статуса/CTA «включить» в Профиле, не объясняется, что пуши = движок поддержки.
7. **Асимметрия лога:** у аллергена one-tap «Дал», у обычного введённого — только полный лист. → свайп «дал сегодня» из каталога.

**P3:** время суток в логе (теряется при бэкдейте), структурная порция, аффорданс «изменить» (нет шеврона), подтверждение удаления записи, кастомный продукт без категории/группы аллергена (кастомный аллерген не попадает в трекер!), бесполезное поле «возраст».

## Что перенять (диари-совместимое)

- **PDF/share «для педиатра»** — дёшево, виральность, Pro-хук.
- **Быстрый лог из сетки иконок + крупные таргеты** (Huckleberry).
- **Богаче поля лога:** количество / «понравилось» / способ приготовления (Nara).
- **Трекер стула ↔ продукт** (Avo, Free to Feed) — нейтральный sticky-хук.
- **Фото к реакции** (Free to Feed).
- **Список покупок из СВОИХ планов** (не «купи рекомендованное»).
- **Монетизация:** ядро-дневник + вся killer-петля **бесплатно** (воронка). Pro = PDF + мульти-ребёнок + шеринг/синк + аналитика. Обязательно **lifetime** рядом с годовым (RU любит one-time). Не гейтить движок удержания.

⚠️ Развилка: «реакция авто-двигает статус» (как Baby Allergy Tracker) — **осознанно НЕ делаем** (запись + стоп вручную; кейс SpoonfulOne подтверждает безопасность). Максимум — описательный ярлык «была реакция», без авто-остановки.

## Рекомендованный роадмап (диари-safe)

**Сейчас:** ① вытащить планирование на поверхность; ② блок «Сегодня записано/запланировано» на главной (он же редизайн блока ребёнка #24); ③ «Дал» с выбором продукта + бэкдейт; ④ секция уведомлений в Профиле; ⑤ онбординг: имя опц. + «Назад» + группировка «уже ввели».
**Дальше:** PDF-экспорт (P1 + первый Pro-хук); быстрый лог свайпом; богаче поля лога; кастомный продукт (категория+группа аллергена).
**Бигбеты/v2:** мульти-кейгивер синк (или явный контр-питч «приватность/офлайн»), трекер стула, фото, Watch/виджеты, монетизация.

Источники — в транскриптах разведок (Solid Starts, Baby Allergy Tracker, Avo, Huckleberry, Nara, Baby Daybook, TryBaby, FeedYou, «Мой прикорм», Wachanga и др.).

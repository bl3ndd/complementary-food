- [ ] **Privacy Policy URL** = `https://<домен>/privacy.html`
- [ ] **Support URL** = сайт / страница поддержки (1.5)
- [ ] **Возрастной рейтинг 4+** (контент; в Kids Category НЕ вступаем)
